## Descrição do Projeto
Este projeto foi desenvolvido por mim, estudante do 2º semestre de Análise e Desenvolvimento de Sistemas (ADS), com o objetivo de praticar e aplicar conceitos de **HTML, CSS e JavaScript** em um projeto criativo.

## Como foi feito
**HTML (index.html)**  
  Responsável pela estrutura da página, onde foram definidos os elementos principais como título, vídeo da câmera, canvas para os efeitos e o áudio do Chidori.

- **CSS (style.css)**  
  Utilizado para estilizar toda a interface.  
  Adicionei um **wallpaper de fundo**.  
  Criei efeitos de **gradiente animado** no título.  
  Estilizei as **bordas da câmera** com brilho e animação.  
  Ajustei posicionamento e responsividade dos elementos.

- **JavaScript (script.js)**  
  Responsável pela lógica do projeto.  
  Integração com a **câmera** usando MediaPipe Hands.  
  Renderização dos efeitos elétricos no **canvas**, simulando o Chidori.  
  Controle do **áudio** para dar imersão ao efeito.  
  Interação em tempo real com a mão detectada.

###  Objetivo
O projeto recria o efeito **Chidori** do anime Naruto, permitindo que qualquer pessoa abra o site, permita acesso à câmera e veja o efeito elétrico em tempo real na própria mão.

### Passo a passo para acessar e testar:
1. Clique no link abaixo para abrir o site:
   isaadsl.github.io/ChidoriHtml/

3. O navegador vai pedir **permissão para acessar a câmera**.  
   - Clique em **Permitir**.

4. Pronto! O efeito **Chidori** aparece em tempo real na sua mão, junto com som e visual elétrico.


### Observações importantes
- Funciona melhor no **Google Chrome** ou **Microsoft Edge**.  
- É necessário ter uma **câmera** conectada ao computador.  
- Se negar a permissão da câmera, o efeito não aparece.  
- Caso queira testar novamente, vá nas configurações do navegador e libere o acesso à câmera para o site.
